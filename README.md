Happy Birthday site for Mommy

Place your MP3 file as `assets/music.mp3` (the page will try to play that file when the visitor clicks the Enter button).

To preview locally:

PowerShell / Command Prompt:

```powershell
# from the project folder
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Or just open `index.html` in your browser (clicking Enter is required to start audio in most browsers).
